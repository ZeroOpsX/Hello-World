# Hello World

A small Python project demonstrating a simple `print`-based program.

## Description

This repository contains a beginner-friendly Python script that prints a greeting message and introduces a cybersecurity learning journey.

## Files

- `main.py` - The Python script that prints the output.
- `README.md` - Project overview and instructions.

## Getting Started

1. Make sure Python is installed on your computer.
2. Open a terminal in this project folder.
3. Run the script with:

```bash
python main.py
```

## Output

The script prints:

- `Hello, Cybersecurity World!`
- `I am starting my journey to become a security professional.`

## Notes

This project is a useful first step for learning Python and getting comfortable with running scripts.
